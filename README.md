bot插件扩展，用于快速更新语录库

## 如何使用

- 安装并运行pm2
```
cd /root/hitokoto-api
npm i -g pm2
pm2 startup
pm2 start ecosystem.config.js --env production
pm2 save
```

- 运行程序

  `python3 scu.py`
